# nday

An exeptionally simple CLI program written in Rust to automate my daily notes

Creates or Opens a note for everyday with the format:

```markdown
MONTH DAY

To-do today:
-

Done today:
-

Kicked to tomorrow:
-

```

Uses `vim` to edit notes
